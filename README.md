# Portfólio | Mirer Balbino

Portfólio pessoal de Mirer Balbino de Andrade Junior, Desenvolvedor Backend
Pleno, com foco em APIs, integrações, sistemas de negócio, automações
operacionais e cases profissionais.

## Stack

- Vue 3
- Vite
- Tailwind CSS
- Font Awesome
- Prettier

## Como Rodar

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev`: inicia o servidor local.
- `npm run build`: gera a versão de produção.
- `npm run preview`: visualiza o build localmente.
- `npm run format`: valida a formatação com Prettier.
- `npm run format:fix`: aplica a formatação com Prettier.
